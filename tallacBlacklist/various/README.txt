Building and running Tallac Blacklist
--------------------------------------------

The application can be run using Ant. The Ant
build file (build.xml) includes the following build targets:

  - 'help' which demonstrates the available targets.
  - 'clean' which removes generated files.
  - 'run' which only compiles the example application.
